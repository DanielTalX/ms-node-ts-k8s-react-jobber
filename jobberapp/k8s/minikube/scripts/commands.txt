## Build docker image
docker build -t my_dockerhub_id/image_name .

## Tag docker image
docker tag my_dockerhub_id/image_name my_dockerhub_id/image_name:new_tag

## Push to docker hub
docker push my_dockerhub_id/image_name:new_tag
